# n8n Study

n8n 학습 프로젝트입니다.

## Python Jupyter Notebook 환경

### 환경 설정

1. 가상환경 생성 및 활성화:
```bash
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
```

2. 패키지 설치:
```bash
pip install -r requirements.txt
```

3. Jupyter Notebook 실행:
```bash
jupyter notebook
```

또는 JupyterLab 사용:
```bash
jupyter lab
```

### 파일 구조

- `code_review.ipynb`: 코드 리뷰를 위한 노트북
- `requirements.txt`: Python 패키지 의존성
